# E-commerce-sales-analysis

## Project Overview
This project analyzes an e-commerce dataset to understand sales patterns, product performance, and regional revenue trends.

## Tools Used
Python
Pandas
Matplotlib
Seaborn

## Dataset
Superstore Sales Dataset

## Key Analysis
- Sales by region
- Monthly revenue trends
- Top performing products
- Category performance

## Key Insights
- West region generates the highest revenue
- Technology category contributes the most sales
- Sales peak during the holiday season
